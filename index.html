HICHORD TEST PROCEDURE

PRODUCTION QUALITY ASSURANCE PROTOCOL

REQUIRED EQUIPMENT: USB-C cable, headphones (3.5mm), MIDI device (optional), speaker/audio input device
01

CHARGING INDICATOR

PROCEDURE

  1. Locate USB-C port on side of HiChord
  2. Connect USB-C cable to HiChord and power source
  3. Observe LED on side panel

EXPECTED RESULT

● RED LED: Charging in progress
● BLUE LED: Fully charged
02

POWER ON SEQUENCE

PROCEDURE

  1. Press and hold power button for 1 second
  2. Observe OLED display boot sequence
  3. Verify firmware version display

EXPECTED RESULT

  • Display shows "HICHORD" title
  • Version "REV 1.95" appears below
  • Boot animation completes smoothly
03

VOLUME CONTROL

PROCEDURE

  1. Locate volume slider on top edge of device
  2. Press any chord button (1-7) to generate sound
  3. Move slider from minimum to maximum
  4. Observe OLED display volume indicator

75%
Volume display (example at 75%)

EXPECTED RESULT

  • Volume bar appears on OLED when slider moved
  • Audio level changes smoothly with slider position
  • No crackling or distortion at any volume level
04

FUNCTION BUTTONS

PROCEDURE

  1. Press F1 (gray) button - Should open Settings menu
  2. Press F2 (yellow) button - Should open Sounds/Effects menu
  3. Press F3 (red) button - Should open Mode/Tempo menu
  4. Press each button again to exit respective menu

  • Correct menu appears for each button
  • No stuck buttons or double-triggers

06

JOYSTICK BUTTON PRESS

PROCEDURE

  1. Press F2 to enter Sounds menu
  2. Navigate to a sound using joystick directions
  3. Press down on joystick (click) to select sound
  4. Verify selection is confirmed

EXPECTED RESULT

  • Joystick click registers as button press
  • Selection is confirmed with visual feedback
  • Button has tactile click feel
07

BUILT-IN SPEAKER

PROCEDURE

  1. Ensure no headphones or USB cable connected
  2. Set volume to 50%
  3. Press chord buttons 1-7 sequentially
  4. Listen for clear audio output from speaker

EXPECTED RESULT

  • Clear audio from internal speaker
  • No distortion, crackling, or buzzing
  • All 7 chord buttons produce sound
  • Speaker has adequate volume and clarity
08

HEADPHONE OUTPUT

PROCEDURE

  1. Connect headphones to 3.5mm jack on right side
  2. Verify internal speaker mutes automatically
  3. Press chord buttons 1-7
  4. Listen for stereo audio in headphones

EXPECTED RESULT

  • Internal speaker mutes when headphones connected
  • Clear stereo audio in both left and right channels
  • No ground noise or interference
  • Volume control functions properly
09

USB-C AUDIO OUTPUT

PROCEDURE

  1. Connect HiChord to computer via USB-C cable
  2. Computer should recognize HiChord as audio device (no drivers needed)
  3. Set computer audio output to "HiChord"
  4. Press chord buttons and verify audio on computer
♫ USB
AUDIO MODE
USB connection indicator

EXPECTED RESULT

  • HiChord appears as USB audio device
  • Class-compliant (no driver installation required)
  • Clean digital audio output to computer
  • Low latency, no dropouts
10

MIDI OVER USB-C

PROCEDURE

  1. Connect HiChord to computer via USB-C
  2. Open DAW or MIDI monitor software
  3. Select "HiChord" as MIDI input device
  4. Press chord buttons 1-7
  5. Verify MIDI note messages appear in monitor

EXPECTED RESULT

  • HiChord appears as USB MIDI device
  • MIDI note-on/note-off messages sent correctly
  • Each chord button sends appropriate MIDI notes
  • MIDI timing is accurate
11

BATTERY LEVEL INDICATOR

BATCH 2+ ONLY

PROCEDURE

  1. Disconnect USB-C cable (run on battery)
  2. Observe top-right corner of OLED display
  3. Battery percentage should be visible
  4. Connect USB-C cable and verify charging indicator appears
85%
Battery indicator (top-right of display)

EXPECTED RESULT

  • Battery percentage displays correctly
  • Level updates as battery drains/charges
  • Low battery warning appears below 20%
  • Charging icon appears when USB connected
NOTE: Skip this test for Batch 1 units (no battery detection)
12

MICROPHONE INPUT

BATCH 4+ ONLY

PROCEDURE

  1. Hold F3 + press Chord Button 6 to enter MIC_SAMPLE mode
  2. Follow on-screen instructions to record a sample
  3. Speak or play a note into microphone
  4. Press joystick to stop recording
  5. Press chord button to play back recorded sample
● REC
Recording in progress

EXPECTED RESULT

  • Microphone captures audio clearly
  • Recorded sample plays back correctly
  • Sample can be played chromatically across chord buttons
  • No excessive noise or interference
NOTE: Skip this test for Batch 1-3 units (no microphone hardware)
